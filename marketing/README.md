# Sextant marketing materials

Everything needed to submit Sextant to the App Store, plus a head start on Google Play and the website.

```
marketing/
├── app-store-listing.md   # All App Store Connect form fields
├── privacy-policy.md      # Template to publish at boringapps.net/sextant/privacy
├── screenshots-plan.md    # The intended carousel, capture commands, status-bar setup
└── screenshots/
    ├── iphone/            # 1320×2868 — App Store 6.9" required
    └── ipad/              # 2064×2752 — App Store 13" required
```

## Submission checklist

1. `app-store-listing.md` — copy/paste each field into App Store Connect.
2. `privacy-policy.md` — host the rendered HTML at the privacy-policy URL listed in the listing.
3. `screenshots/` — upload to the matching device class in App Store Connect.

Open items live at the bottom of `app-store-listing.md`.
